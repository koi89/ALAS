# Atajos de teclado

Referencia rápida de todos los atajos de teclado de ALAS.

---

## Archivo

| Acción | Atajo |
|--------|-------|
| Abrir archivo | `Ctrl+O` |
| Abrir múltiples archivos | `Ctrl+Shift+O` |
| Guardar proyecto | `Ctrl+S` |
| Cargar proyecto | `Ctrl+Shift+S` |
| Exportar | `Ctrl+E` |
| Salir | `Ctrl+Q` |

---

## Vista

| Acción | Atajo |
|--------|-------|
| Restablecer cámara | `R` |
| Vista superior | `T` |
| Vista frontal | `F` |
| Vista lateral | `S` |
| Pantalla completa | `F11` |

---

## Procesamiento

| Acción | Atajo |
|--------|-------|
| Clasificar terreno | *(solo menú)* |
| Historial de clasificaciones | `Ctrl+Shift+H` |
| Procesamiento por lotes | `Ctrl+B` |
