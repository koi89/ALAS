# Keyboard Shortcuts

Quick reference for all ALAS keyboard shortcuts.

---

## File

| Action | Shortcut |
|--------|----------|
| Open file | `Ctrl+O` |
| Open multiple files | `Ctrl+Shift+O` |
| Save project | `Ctrl+S` |
| Load project | `Ctrl+Shift+S` |
| Export | `Ctrl+E` |
| Quit | `Ctrl+Q` |

---

## View

| Action | Shortcut |
|--------|----------|
| Reset camera | `R` |
| Top view | `T` |
| Front view | `F` |
| Side view | `S` |
| Fullscreen | `F11` |

---

## Processing

| Action | Shortcut |
|--------|----------|
| Classify terrain | *(menu only)* |
| Classification history | `Ctrl+Shift+H` |
| Batch processing | `Ctrl+B` |

---

## Analysis

| Action | Shortcut |
|--------|----------|
| My Reports | `Ctrl+Shift+R` |

---

## Tools

| Action | Shortcut |
|--------|----------|
| Measurement history | `Ctrl+H` |
| Cancel active tool | `Esc` |

---

## Application

| Action | Shortcut |
|--------|----------|
| Settings | `Ctrl+,` |
